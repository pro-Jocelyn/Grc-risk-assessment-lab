# Executive Summary

## Project Title
GRC Risk Assessment Lab: HIPAA & NIST Control Mapping

## Purpose
The purpose of this lab was to perform a basic GRC risk assessment for a fictional small dental clinic that stores electronic patient information.

## Scope
The assessment focused on patient records, employee access, phishing, device security, Wi-Fi, backups, and cloud file storage.

## Key Findings
The highest risks identified were unauthorized access to patient records, phishing attacks, lost or stolen devices, former employee access, and untested backups.

## Recommendations
The dental clinic should implement multi-factor authentication, role-based access, device encryption, security awareness training, backup testing, automatic screen locks, and formal user access removal procedures.

## Conclusion
This project demonstrates foundational GRC skills including risk identification, risk scoring, HIPAA safeguard alignment, NIST control mapping, and executive reporting.
