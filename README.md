# GRC Risk Assessment Lab: HIPAA & NIST Control Mapping

## Project Overview
This project is a beginner-friendly GRC risk assessment lab based on a fictional small dental clinic that stores patient information electronically.

The goal of this lab is to identify cybersecurity and privacy risks related to patient records, employee access, phishing, and device security, then map those risks to HIPAA safeguards and NIST security controls.

## Scenario
A small dental clinic stores patient information electronically and wants to reduce cybersecurity and privacy risks related to patient records, employee access, phishing, and device security.

## Skills Demonstrated
- Risk assessment
- Risk register creation
- HIPAA safeguard identification
- NIST control mapping
- Compliance documentation
- Executive summary writing

## Files Included
- Risk Register
- Control Mapping Matrix
- Executive Summary

## Frameworks Used
- HIPAA Security Rule
- NIST SP 800-53
- NIST Cybersecurity Framework
