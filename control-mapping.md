# Control Mapping Matrix

| Risk ID | Risk Summary | HIPAA Safeguard | NIST Control | Control Name | Recommendation |
|---|---|---|---|---|---|
| R-001 | Unauthorized access to patient records | Technical Safeguard | AC-2 / IA-2 | Account Management / Identification & Authentication | Enable MFA, unique user accounts, and role-based access |
| R-002 | Lost or stolen device exposes patient data | Technical Safeguard | SC-28 | Protection of Information at Rest | Use full-disk encryption on laptops and workstations |
| R-003 | Phishing leads to credential compromise | Administrative Safeguard | AT-2 | Awareness Training | Provide security awareness and phishing training |
| R-004 | Former employee keeps access | Administrative Safeguard | AC-2 | Account Management | Disable accounts immediately after termination |
| R-005 | Unauthorized Wi-Fi access | Technical Safeguard | AC-18 | Wireless Access | Use secure Wi-Fi settings and separate guest network |
| R-006 | Ransomware impacts patient record availability | Administrative Safeguard | CP-9 / CP-10 | System Backup / Recovery | Perform and test regular backups |
| R-007 | Unlocked screen exposes patient data | Physical/Technical Safeguard | AC-11 | Session Lock | Enable automatic screen locking |
| R-008 | Cloud files shared incorrectly | Technical Safeguard | AC-3 | Access Enforcement | Restrict file sharing and review permissions regularly |
